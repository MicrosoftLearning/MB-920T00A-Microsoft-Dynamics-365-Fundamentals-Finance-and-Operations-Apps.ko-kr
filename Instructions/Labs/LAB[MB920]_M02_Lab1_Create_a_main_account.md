# 학습 경로 2: Microsoft Dynamics 365 Finance의 기본 사항 파악
# 모듈 2: 총계정 원장 설명

## 랩 2.1: 기본 계정 만들기

## 랩 설정

   - **예상 시간:** 5분

## 목표

이 랩에서 수행하는 작업은 다음과 같습니다.

1. 법인을 만듭니다.

2. 회계 달력을 만듭니다.

3. 계정 차트를 만듭니다.

4. 계정 차트에서 주 계정을 만듭니다.

5. 유형 부서 및 비용 센터의 운영 단위를 만듭니다.

6. 주 계정 및 비용 센터를 사용하여 회계 구조를 만듭니다.

7. 원장을 구성합니다.

# 랩 단계

## 법인 만들기

1. Dynamics 365 Finance의 왼쪽 탐색 창에서 **모듈****&gt;****조직 관리****&gt; 조직 &gt; 법인**을 선택합니다.

2. **법인** 페이지의 작업 창에서 **새로 만들기** 단추를 선택하여 새 법인을 만들고 다음 정보를 입력합니다.

    - 이름: **Contoso 데모 시스템**

    - 회사: **CDS**

    - 국가/지역: **USA**

3. 하단에서 **확인**을 선택합니다.

## 회계 달력 만들기

1. Dynamics 365 Finance의 왼쪽 탐색 창에서 **모듈****&gt;****총계정원장****&gt; 달력 &gt; 회계 달력**을 선택합니다. 

2. **회계 달력** 페이지의 작업 창에서**새 달력** 단추를 선택하여 새 회계 달력을 만들고 다음 정보를 입력합니다.

    - 달력: **CDS**

    - 설명: **데모 시스템 달력**

    - 회계 연도 시작: **2024년 9월 30일**

    - 회계 연도 종료: **2025년 10월 1일**

    - 회계 연도 이름: **2024-25**

    - 기간: **1**

    - 단위: **월**

3. **생성** 단추를 선택합니다.

시스템은 개시 기간 1개와 마감 기간 1개, 12개월 해당하는 12개 기간을 포함하여 14개의 기간을 생성합니다.

## 계정 차트 만들기

1. Dynamics 365 Finance의 왼쪽 탐색 창에서 **모듈** > **총계정원장****&gt;****계정 차트****&gt; 계정 &gt; 계정 차트**를 선택합니다.

2. **계정 차트** 페이지의 작업 창에서 **새로 만들기** 단추를 선택하여 새 계정 차트를 만들고 다음 정보를 입력합니다.

    - **계정 차트**: 데모 시스템

    - **설명**: Contoso 데모 시스템

3. **기본 계정** FastTab에서 **새로 만들기** 단추를 선택하여 다음 값으로 기본 계정을 만듭니다.

    - 기본 계정: **1000**

    - 이름: **현금**

    - 기본 계정 유형: **대차 대조표**

    - 기본 계정 범주: **현금**

    - DB/CR 기본값: **직불**

4. 다른 기본 계정 만들기:

    - 기본 계정: **3000**

    - 이름: **수익**

    - 기본 계정 유형: **수익**

    - 기본 계정 범주: **REV**

    - DB/CR 기본값: **크레딧**

5. 다른 기본 계정 만들기:

    - 기본 계정: **6000**

    - 이름: **여행 경비**

    - 기본 계정 유형: **경비**

    - 기본 계정 범주: **EXP**

    - DB/CR 기본값: **직불**

##  운영 단위 생성

1. **모듈****&gt;****조직 관리****&gt; 조직 &gt; 운영 단위**로 이동합니다.

2. 작업 창에서 **새로 만들기** 버튼을 선택한 다음 **운영 중인 부서** 단위 유형을 선택하고 다음 값을 입력합니다.

    - 이름: **CDS_Training**

3. 작업 창에서 **새로 만들기** 버튼을 선택한 다음 **비용 센터 운영** 단위 유형을 선택하고 다음 값을 입력합니다.

    - 이름: **CDS_Purchase**

4. 비용 센터를 두 개 더 추가합니다: **CDS_IT** 및 **CDS_Admin**.

## 계정 구조 만들기

1. **재무 및 운영 홈** 페이지의 오른쪽 상단에서 작업을 수행할 회사가 **CDS**인지 확인합니다.

2. 필요한 경우 회사를 선택하고 메뉴에서 **CDS**를 선택합니다.

3. Dynamics 365 Finance의 왼쪽 탐색 창에서 **모듈** > **총계정 원장****&gt;****계정 차트 &gt; 구조 &gt; 계정 구조 구성**을 선택합니다.

4. 작업 창에서 **새로 만들기** 버튼을 선택하여 다음 값으로 새 계정 구조를 만듭니다.

    - 계정 구조: **CDS_BS**

    - 설명: **CDS 대차 대조표**

    - 기본 계정 추가: **예**

5. **생성** 단추를 선택합니다.

6. **세그먼트 및 허용된 값** 빠른 탭에서 **추가** 버튼을 선택하여 **기본 계정 범위 1000...2999**를 추가합니다.

7. 작업 창에서 **활성화** 버튼을 선택한 다음 일괄 처리 대화 상자에서 **활성화** 버튼을 선택합니다.

8. **계정 구조** 페이지의 작업 창에서 **새로 만들기** 버튼을 선택하여 다음 값으로 다른 계정 구조를 만듭니다.

    - 계정 구조: **CDS_Revenue**

    - 설명: **CDS 매출**

    - 기본 계정 추가: **예**

9. **생성** 단추를 선택합니다.

10. **세그먼트 및 허용 값** 빠른 탭에서 **추가** 버튼을 선택하여 **기본 계정 범위 3000...5999**를 추가합니다.

11. 작업 창에서 **활성화** 버튼을 선택한 다음 일괄 처리 대화 상자에서 **활성화** 버튼을 선택합니다.

12. **계정 구조** 페이지의 작업 창에서 **새로 만들기** 버튼을 선택하여 다음 값으로 다른 계정 구조를 만듭니다.

    - 계정 구조: **CDS_Expense**

    - 설명: **CDS 비용**

    - 기본 계정 추가: **예**

13. **생성** 단추를 선택합니다.

14. **세그먼트 및 허용된 값** 빠른 탭에서 **추가** 버튼을 선택하여 **기본 계정 범위 6000..9999**를 추가합니다.

15. **세그먼트 및 허용된 값** 빠른 탭에서 **세그먼트 추가** 버튼을 선택합니다.

16. **세그먼트 추가** 대화 상자에서 **CostCenter**를 선택한 다음 **세그먼트 추가** 버튼을 클릭합니다.

17. **CostCenter** 필드에 **253..255**를 입력합니다. 

18. 작업 창에서 **활성화** 버튼을 선택한 다음 일괄 처리 대화 상자에서 **활성화** 버튼을 선택합니다.

## 원장 구성

1. **재무 및 운영 홈** 페이지의 오른쪽 상단에서 작업을 수행할 회사가 **CDS**인지 확인합니다.

2. 필요한 경우 회사를 선택하고 메뉴에서 **CDS**를 선택합니다.

3. Dynamics 365 Finance의 왼쪽 탐색 창에서 **모듈** > **총계정원장 &gt; 원장 설정 &gt; 원장** 페이지를 선택하고 다음을 구성합니다.

    - 계정 차트: **데모 시스템**

    - 회계 달력: **CDS**

    - 회계 통화: **USD**

    - 보고 통화: **USD**

    - 회계 통화 환율 유형: **기본값**

    - 예산 환율 유형: **예산**

4. **계정 구조** FastTab에서 **추가** 단추를 선택하여 계정 구조 **CDS_BS**를 추가합니다.

> 필요한 경우 계정 구조를 추가하려면 **예** 선택합니다.

5. **CDS_Revenue** 및 **CDS_Expense** 두 개의 계정 구조를 더 추가합니다.

그러면 총계정원장 모듈의 기본 구성이 완료됩니다. 이제 저널 구성을 계획하여 저널을 게시할 수 있습니다.
